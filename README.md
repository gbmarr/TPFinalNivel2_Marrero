# Aplicación de Escritorio para Gestión de Inventario
Esta aplicación de escritorio ha sido desarrollada utilizando C# en .NET Framework. Permite gestionar un inventario de productos almacenados en una base de datos SQL Server. Los elementos gráficos disponibles en la Toolbox de .NET se utilizan para crear una interfaz de usuario intuitiva y funcional.

# Tecnologías Utilizadas
C# y .NET Framework: Utilizados en el backend para la lógica de la aplicación y la conexión con la base de datos.
Elementos Gráficos de .NET: Utilizados para crear la interfaz de usuario, incluyendo controles como botones, cuadros de texto, tablas, etc.
SQL Server: Base de datos utilizada para almacenar y gestionar los datos del inventario.
ADO.NET: Utilizado para la conexión y manipulación de datos en la base de datos SQL Server.

# Funcionalidades Principales
La aplicación ofrece las siguientes funcionalidades:
- Visualización de Inventario: Muestra una lista de productos disponibles en el inventario, con detalles como nombre, descripción, marca, categoría, precio, etc.

- Búsqueda de Artículos: Permite realizar búsquedas de artículos por distintos criterios, como nombre, marca, categoría, etc.

- Agregar, Modificar y Eliminar Artículos: Los usuarios pueden agregar nuevos productos al inventario, modificar la información de productos existentes y eliminar productos que ya no estén disponibles.

- Visualización de Detalles: Permite ver detalles adicionales de cada producto, como la cantidad en stock, la fecha de ingreso, etc.
